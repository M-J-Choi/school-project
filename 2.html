프로그래밍 언어

프로그래밍 언어란 컴퓨터 시스템을 구동시키키는 소프트웨어를 작성하기 위한 형식언어로 고급언어로 갈수록 사람이 사용하는 언어에 가깝습니다.

또한 프로그래밍 언어들은 엄청나게 많아서 제가 관심이 있는 웹 프로그래밍 분야에서 가장 많이 사용되는 언어 3가지를 알려드리도록 하겠습니다.

<프로그래밍 언어의 종류>

1.HTML: HTML은 HyperText Markup Language의 약자입니다. 각각의 HTML 태그는 웹 페이지의 디자인이나 기능을 결정하는데 사용됩니다.

2.CSS: CSS는 Cascading Style Sheets의 약자입니다. CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있습니다.

3.JavaScript: JavaScript는 객체(object) 기반의 스크립트 언어입니다. JavaScript로는 웹의 동작을 구현 할 수 있습니다.
